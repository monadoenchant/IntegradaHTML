A ciência de dados é uma área da computação, ligada diretamente ao avanço de inteligência artificial, que analisa padrões e ações em um campo específico para tomar decisões eficientes para aquele campo.
A ciência de dados é conectada também a coleta de dados, que pode ser voluntária ou involuntária ao usuário, o que tende a levantar algumas preocupações éticas sobre o campo como um todo.
No mercado, ciência de dados se provou ser muito útil, mas cabe aos engenheiros de software que à aplicam saber tomar as decisões mais humanas com respeito às informações que certo programa coleta.

(Simbolo utilizado para representar Base de Dados)
